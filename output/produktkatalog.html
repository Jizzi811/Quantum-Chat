01 · GRUNDLAGEN

Was ist Quantum?

Quantum ist ein KI-Wesen, das aus deinen Gedanken funktionierende Dinge baut. Im Zentrum steht ein Chat-Fenster — drumherum gruppieren sich Studios für Bild, Video, Sprache und Spiele sowie ein System aus Skills und Automationen, die Quantum eigenständig für dich arbeiten lassen.

💬
KI-Chat
Antworten live vom konfigurierten KI-Modell — das Herzstück von Quantum.
🎨
Bild-Studio
Bilder aus Text generieren, per Gemini/Imagen.
🎬
Video-Studio
Remotion-Videos direkt per Prompt erzeugen.
🔊
TTS Studio
Text zu Sprache mit VoxCPM & eigenem Stimm-Design.
🎮
Game Studio
Spielbare Browser-Games per Prompt bauen.
⟳
Automationen
Trigger & Aktionen einrichten — ganz ohne Code.

Songsee
Audio direkt im Browser als farbiges Spektrogramm sichtbar machen.

02 · STUDIOS

Die Studios im Überblick

Jedes Studio ist auf eine Aufgabe spezialisiert. Öffne es über sein Panel-Icon und beschreibe in eigenen Worten, was entstehen soll.

🎨
Bild-Studio Gemini / Imagen
Beschreibe ein Bild in Worten — Motiv, Stil, Stimmung, Farben — und Quantum generiert es.
„Halte Bildideen konkret: Motiv, Stil, Licht, Perspektive.“
🎬
Video-Studio Remotion
Aus einem Prompt entsteht ein animiertes Video-Kurzformat, Szene für Szene aufgebaut.
„Beschreibe die Szenenfolge wie ein kleines Drehbuch.“
🔊
TTS Studio VoxCPM
Text wird zu natürlicher Sprache — inklusive eigenem Stimm-Design für Tonfall und Charakter.
„Gib Tonfall und Tempo mit an: ruhig, energisch, erzählend …“
🎮
Game Studio Browser-Games
Aus einer Spielidee entsteht ein direkt spielbares Browser-Game.
„Nenne Spielprinzip, Steuerung und Ziel in einem Satz.“
🌊
Songsee Lokale Analyse
Verwandelt eine Audiodatei direkt im Browser in ein farbiges Spektrogramm — nichts wird hochgeladen.
„Wähle einen Farbstil und grenze bei Bedarf Start & Dauer ein.“
03 · SKILLS

Skills — Werkzeuge im Chat

Skills sind kleine, sofort einsatzbereite Funktionen direkt im Chat. Rufe sie mit /skill <name> <eingabe> auf oder klicke sie im Skills-Panel an. Jeder Skill lässt sich dort einzeln ein- und ausschalten.

🧮 Neuro-Rechner /skill rechner
Rechnet mathematische Ausdrücke inklusive Klammern und Potenzen.
» /skill rechner 12*(3+4)^2
« ⚡ Ergebnis: 588
💡 Ideen-Generator /skill ideen
Wirft dir spontan eine Produkt- oder Projektidee zu.
» /skill ideen
« 💡 Idee: Ein Habit-Tracker, der Gewohnheiten als leuchtende Neon-Ringe zeigt
📝 Text-Kompressor /skill zusammenfassen
Fasst eingefügten Text auf die zwei zentralen Kernsätze zusammen.
» /skill zusammenfassen <Text>
« Kurzfassung (6 Sätze → 2): » …
🔐 Passwort-Schmiede /skill passwort
Erzeugt ein starkes, zufälliges Passwort — rein lokal im Browser.
» /skill passwort 20
« 🔐 xT9!kLm2Qz…

🔧 SKILLS EIN-/AUSSCHALTEN
Im Skills-Panel hat jede Karte einen Schalter. Deaktivierte Skills bleiben sichtbar, antworten im Chat aber nicht mehr — praktisch, um das Angebot für dich persönlich zu verschlanken.
04 · AUTOMATIONEN

Quantum arbeitet für dich

Automationen führen eine Aktion (meist einen Skill) automatisch aus — zeitgesteuert oder wenn ein Ereignis eintritt. Erstellt werden sie im Übersicht-Panel: Name vergeben, Auslöser wählen, Aktion + Eingabe festlegen, fertig.

⏱ Alle X Minuten
Löst die Aktion in einem festen Zeitintervall aus.
🕘 Täglich um Uhrzeit
Führt die Aktion einmal am Tag zur gewählten Uhrzeit aus.
💬 Wenn Nachricht Stichwort enthält
Reagiert, sobald ein bestimmtes Wort im Chat auftaucht.
🚀 Bei jedem Start
Läuft automatisch, sobald Quantum geöffnet wird.
◫ FERTIGE AUTOMATIONS-REZEPTE
Täglich 09:00 Zeit-Kristall zeigt den Countdown zum nächsten Meilenstein.
Alle 60 Min Hype-Modul schickt einen Motivationsspruch in den Chat.
Stichwort „Idee“ Ideen-Generator wirft automatisch einen neuen Vorschlag zu.
Bei jedem Start Passwort-Schmiede hält ein frisches Passwort bereit.
05 · WORKFLOW

So chattest du richtig mit Quantum

Vier Gewohnheiten, mit denen du in Sekunden bessere Ergebnisse bekommst.

1
Sei konkret
Ziel, Ton und Format in einem Satz nennen — je klarer die Vorgabe, desto treffender das Ergebnis.
2
Nutze /skill gezielt
Für feste Aufgaben (rechnen, würfeln, zusammenfassen) direkt den passenden Skill aufrufen statt frei zu fragen.
3
Automatisiere Wiederholungen
Alles, was du dreimal manuell gemacht hast, verdient eine Automation.
4
Wechsle bewusst das Studio
Für Bild, Video, Sprache oder Spiele das passende Studio öffnen statt alles über den Chat zu erzwingen.
🔑 ZUGANG
Ohne Zugangscode läuft Quantum im lokalen Demo-Modus (ohne Live-KI-Antworten). Mit Abo oder Code erhältst du vollen Zugang zu Chat, allen Skills sowie Bild-, Video- und Sprach-Studio.
